# MinervaApi

This gem manages Open Access Papers Archives

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'minerva_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minerva_api

## Usage

See MinervaApi::Arxiv for details
